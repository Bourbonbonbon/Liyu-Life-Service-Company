# 里御生活服务公司网站管理系统

这是一个功能完整的公司网站管理系统，支持公告发布、品牌介绍、门店信息管理等功能，所有数据都保存在本地（localStorage）。

## 功能特点

### 1. 标签页导航
- 公告管理：发布和展示公司公告
- 品牌介绍：展示品牌信息和介绍
- 门店清单：管理所有门店信息
- 设置：自定义LOGO和背景
- 戳戳我：联系方式管理

### 2. 核心功能
- **图片上传与管理**：支持多图片上传（最多5张）
- **本地数据存储**：所有数据保存在浏览器localStorage中
- **密码保护**：管理操作需要密码验证
- **响应式设计**：适配各种屏幕尺寸
- **图片放大查看**：点击图片可全屏查看

### 3. 管理功能
- 添加、修改、删除各项内容
- 自定义LOGO和背景图片
- 所有操作都有密码保护

## 数据保存
所有数据都保存在浏览器的localStorage中，这意味着：
- 数据不会上传到服务器
- 数据只在当前浏览器中有效
- 清除浏览器缓存会删除所有数据

## 注意事项
1. 建议定期备份重要数据
2. 在不同浏览器或设备上需要重新设置
3. 图片以Base64格式存储，大量图片可能占用较多存储空间

## 浏览器兼容性
支持所有现代浏览器（Chrome、Firefox、Safari、Edge等）。

这个系统完全在浏览器中运行，无需服务器支持，非常适合小型公司或个人使用。
